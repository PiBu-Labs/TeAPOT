# TeAPOT

**TeAPOT** (Terrain-Aligned Positioning and Orientation Toolkit) is an outdoor mixed reality system that uses a smartphone running the [SPOT](https://github.com/PiBu-Labs/SPOT) app (Spatial Positioning and Orientation Tool) to transfer GNSS position and true heading to a standalone headset through AprilTag-based alignment. TeAPOT then reconstructs a local 2.5D terrain surface from headset depth data for terrain-aligned geospatial visualization.
